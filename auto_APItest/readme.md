## 接口自动化测试文档  

### 基于python的requests,使用Pytest框架实现接口自动化  

### 项目目录结构  

* Utils 自定义封装类
   * Requests.py 封装Requests请求方式
   * Email.py 邮件发送封装
   * Constans.py 变量
   * Assert.py 断言
   * Log.py 封装Log
   * Token.py 登录token获取
* Config 配置文件信息
* Datas测试数据
* TestCases 测试用例
* Reports 测试报告文档
* Logs 日志文件
* run.py 主程序入口，执行测试
